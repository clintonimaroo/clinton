# Clinton Imaro - Portfolio

My professional portfolio site built with Next.js.

## Description

This is a portfolio website showcasing my work as a Senior Software Engineer.

## Installation

```bash
# Install dependencies
$ npm install
```

## Running the app

```bash
# Development
$ npm run dev

# Production mode
$ npm run build
$ npm start
```

## Building for production

```bash
# Build the application
$ npm run build
```

## Technologies Used

- Next.js
- React
- TypeScript
- SCSS

## Folder Structure

- `app/` - Next.js app directory
  - `components/` - React components
  - `styles/` - Global SCSS styles
  - `layout.tsx` - Root layout
  - `page.tsx` - Home page
- `public/` - Static files (favicon, robots.txt, sitemap.xml)
